# docs/archive/README.md

## 归档说明
`docs/archive/` 用于存放历史任务记录、早期实现文档和不再作为当前执行入口的材料。

## 当前归档内容
- `docs/archive/tasks/`：历史 `task_*.md` 实施记录。
- `docs/archive/legacy/`：早期需求、任务列表、测试用例清单等历史入口文档。

## 使用边界
- 归档文档默认只读。
- 归档文档可以作为“为什么当时这样做”的追溯材料，但不能作为当前实现和决策的主真相源。
- 当前执行顺序、规则和目录解释以根目录下的 `AGENTS.md`、`project_rules.md`、`architecture_map.md`、`plan.md` 为准。
